Руководство пользователя QuantumVault
QuantumVault — это веб-приложение для безопасного локального шифрования и расшифровки файлов. Все криптографические операции выполняются исключительно в вашем браузере, что гарантирует абсолютную конфиденциальность — ваши данные никогда не отправляются на сервер.

🔒 Как зашифровать файл
Загрузите исходный файл: Нажмите на пунктирную область с надписью «Перетащите файл сюда или выберите на диске» и укажите нужный файл, либо просто перетащите его мышкой в это окно. После успешной загрузки ниже появится имя файла.

Задайте мастер-пароль: В соответствующем поле введите надежный пароль.

Примечание: В целях безопасности пароль должен состоять не менее чем из 8 символов. Для проверки правильности ввода вы можете использовать иконку «глаза» (показать/скрыть пароль).

Запустите шифрование: Нажмите кнопку «Зашифровать».

Сохраните результат: После успешной обработки появится зеленое уведомление и кнопка «Скачать обработанный файл». Нажмите её, чтобы сохранить на устройство защищенную копию файла (к оригинальному названию добавится расширение .enc).

🔓 Как расшифровать файл
Загрузите защищенный файл: Перетащите или выберите ранее зашифрованный файл (с расширением .enc) в область загрузки.

Введите пароль: Укажите тот же самый мастер-пароль, который использовался при создании этого зашифрованного файла.

Запустите расшифровку: Нажмите кнопку «Расшифровать».

Сохраните оригинал: Если пароль указан верно, появится кнопка «Скачать обработанный файл». При нажатии на неё исходный файл без расширения .enc будет сохранен на ваш диск. В случае неверного пароля или поврежденного файла система выдаст ошибку.

⚠️ Важные замечания
Отсутствие функции восстановления: Так как процесс шифрования (AES-GCM 256-bit) происходит полностью на вашем устройстве, система нигде не сохраняет ваш мастер-пароль. Если вы забудете или потеряете пароль, восстановить доступ к зашифрованному файлу будет невозможно.

Размер файлов: Приложение работает в оперативной памяти вашего устройства. При попытке зашифровать файлы очень большого размера (например, видео в несколько гигабайт) браузер может зависнуть из-за нехватки ресурсов.
